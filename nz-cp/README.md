### Install

```bash
npm install -g @ngmazamat/nz-cp
bun install -g @ngmazamat/nz-cp
nz-cp --version
```

### Help
```bash
nz-cp --help
```

```bash
usage: nz-cp [OPTIONS] [SOURCE] [DEST]
or: nz-cp [OPTIONS] [OPTION: -t ] [DIRECTORY] [SOURCE]
or: nz-cp [OPTIONS] [SOURCE] [DIRECTORY]

1. Copy SOURCE to DEST
2. Copy SOURCE to DIRECTORY
3. or multiple SOURCE(s) to DIRECTORY.

Arguments:
  directory      directory to list (default: ".")

Options:
  -V, --version  output the version number
```
